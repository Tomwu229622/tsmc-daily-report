## [2026-03-11] — Daily Report v2.0 (PR #1)

### 新增
- HTML 日報產生器 v2.0（`generate_tsmc_html_report.py`）升級為 13 個分析區塊
- **三大法人買賣超**：外資/投信/自營商每日買賣超張數 + 5日走勢圖
- **技術分析**：RSI(14)、MACD、KDJ(9,3,3)、布林通道(20,2)、MA5/20/60/120
- **期貨選擇權**：未平倉量、Put/Call Ratio、最大痛苦點、行使價分布
- **產業生態系股價連動**：14檔（設備/記憶體/封裝/客戶/競爭）
- **法人評級彙整**：Goldman NT$2,330 / JPMorgan NT$2,100 / 共識 NT$2,290
- **財報倒計時**：距下次財報天數、共識EPS、8季超預期率
- **晶圓廠稼動率**：7個製程節點（N2~N28）估算
- **IR 行事曆**：除息日、財報日、技術論壇等重要日期
- **頂端 TOC 導覽列**：fixed 顯示，一鍵跳轉13個區塊
- 每日 HTML 日報：`TSMC_日報_2026-03-09.html`、`TSMC_日報_2026-03-10.html`、`TSMC_日報_2026-03-11.html`
